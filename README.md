# Employee Portal

Internal employee self-service portal built with **React + Vite + Tailwind CSS**.  
All data persisted in **localStorage** — no backend required.

## Features

| Module | Capabilities |
|---|---|
| **Attendance** | Clock In/Out · Late detection (after 9:30 AM) · Filterable history |
| **Daily Progress** | Log tasks · Inline edit · Delete · Grouped by date |
| **Leave Requests** | Apply leave (Sick/Casual/Earned) · Duration counter · Cancel pending |
| **Dashboard** | Stat cards · Recent entries · Quick navigation |

## Setup

```bash
npm install
npm run dev
```
